# PS Portfolio

Park Seojin portfolio prototype built with Next.js, TypeScript, SCSS Modules, GSAP ScrollTrigger, and Lenis.

## Local

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
npm run start
```

## Deploy

### Recommended: GitHub + Vercel

1. Create a GitHub repository, for example `ps-portfolio`.
2. Push this project.
3. Import the repository in Vercel.
4. Framework preset: Next.js.
5. Deploy.

Vercel will give you a live URL like:

```text
https://ps-portfolio.vercel.app
```

You can share that URL and anyone can see the portfolio page directly.

### GitHub Pages?

Possible, but less recommended for this project because it uses Next.js. Vercel is smoother for Next.js routing, build, and previews.
