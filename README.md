# seojin-Portfolio

포트폴리오 제작을 위해 진행 중인 개인 웹 프로젝트입니다.  
프로젝트 중심의 전시형 포트폴리오를 목표로 제작하고 있습니다.

## Links 🔗

- Live Demo: https://psj0406.vercel.app
- Repository: https://github.com/psj0406/ps-portfolio

---

## Tech Stack

- Next.js
- TypeScript
- SCSS Modules
- GSAP
- Lenis
- Vercel

---

## Features

- 프로젝트 중심 포트폴리오 구성
- Hero / Works / Career / Contact 섹션 구조
- 스크롤 애니메이션 적용 예정
- 반응형 레이아웃 적용 예정
